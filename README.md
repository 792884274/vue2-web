# vue2-web
vue2、vue-router2、webpack2框架

想从react转vue，可以跟着我这个项目走，今天只完成了vue框架的搭建，以及路由跳转，当然少不了热更新，webpack服务器等react中有的开发环境，在vue项目中你一样可以用到。
后续我会继续研究vue管理数据的方式，以及非常重要的组件通讯机制。
